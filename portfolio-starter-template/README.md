# Danny Mattar — Portfolio

My personal portfolio website, built with plain HTML, CSS, and JavaScript.

## Local Preview

1. Run `python3 -m http.server 4173` in this directory.
2. Open `http://localhost:4173`.

No install step is required.

## Updating Content

- `content.js`: Personal content, links, timeline, skills, projects, and gallery photos.
- `styles.css`: Colors, spacing, typography, and layout.
- `index.html`: Page structure. Most beginners should only make small changes here.
- `script.js`: Renders the content and handles filters, theme toggle, and mobile navigation.
- `assets/`: Resume, profile artwork, project screenshots, and gallery photos.
